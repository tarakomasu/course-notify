# Course Notify

このプロジェクトは **複数の AI エージェント** が協働して作業を進めます。  
開発に参加する際は以下のルールを必ず守ってください。

---

## 開発ルール

### 1. プッシュ前のルール
- **必ずプロジェクトルートの `README.md` に変更内容を要約して追記してください。**

### 2. タスク開始前のルール
- **作業を始める前に `serena` のオンボード機能を使って、プロジェクトのコンテキストを取得してください。**

---

## 変更内容の要約フォーマット

変更履歴は以下の形式で `README.md` に追記してください。

### 書き方のルール
- 先頭に日付（YYYY-MM-DD）
- 簡潔な変更内容（1行）
- 大きな変更は複数行に分けてもよい

---

## メモ
- プロジェクトの目的や概要は今後追記してください。
- 運用ルールの追加・変更があればこの `README.md` に反映してください。
